Float-based holy grail uses left float, right float, and a center main with matching margins. Old technique but widely supported. Requires clearfix and careful ordering: sidebars must come before main in HTML for floats to work correctly.

Flexbox holy grail uses a flex container on the body row. The main content uses flex: 1 to grow and fill remaining space. Sidebars have fixed widths using flex-shrink: 0. Source order matches visual order, unlike floats.
