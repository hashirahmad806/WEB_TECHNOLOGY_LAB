## How to Run

Open `index.html` in any modern web browser. No server required.

## Coverage

- **display.css / display-exercises.html** — block, inline, inline-block, none, visibility, table layout, navigation, dropdowns
- **float.css / float-exercises.html** — image float, 2-col, 3-col, grid system, clearfix
- **position.css / position-exercises.html** — static, relative, absolute, fixed, sticky, z-index, modal, tooltip
- **flexbox.css / flexbox-exercises.html** — all flex container and item properties, nav, cards, form
- **grid.css / grid-exercises.html** — grid container, item, named areas, span, minmax, auto-fit, auto-fill, fr units
- **media-queries.css / media-query-exercises.html** — breakpoints, mobile-first, orientation, prefers-color-scheme, prefers-reduced-motion, print
- **layouts/** — magazine (float), holy grail (3 methods), dashboard (grid), portfolio (mixed), responsive demo
